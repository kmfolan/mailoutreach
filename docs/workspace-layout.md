# Outbound Forge Workspace Layout

- root `index.html`
- root `styles.css`
- root `app.js`
- `server/src/index.js`
- `server/src/store.js`
- `server/data/db.json`

## Supporting Folders

- `docs` - current product and deployment notes
- `tools` - bundled local runtime and related tooling
- `apps/web` - reserved for a future structured web client
- `apps/mobile` - reserved only if the product eventually needs mobile
- `packages/shared` - reserved for future shared types/helpers
